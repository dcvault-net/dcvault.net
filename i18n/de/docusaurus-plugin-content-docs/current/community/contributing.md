DCVault ist ein Community-Wiki und wächst mit den Leuten, die es nutzen. Jede Seite lässt sich verbessern, und du brauchst dafür kein Konto auf dieser Seite.

## Eine Seite korrigieren oder verbessern

Unten auf jeder Seite gibt es den Link „Diese Seite bearbeiten". Er öffnet die Quelldatei auf GitHub, wo du sie änderst und einen Pull Request stellst. Ein Maintainer schaut drüber und merged. Kleine Korrekturen wie ein Tippfehler, ein toter Link oder ein klarerer Satz sind immer willkommen und so schnell erledigt.

## Etwas Größeres vorschlagen

Für eine neue Seite, eine größere Überarbeitung oder alles, was du vorher besprechen willst, öffne ein Issue auf [GitHub](https://github.com/dcvault-net/dcvault.net) oder starte einen Thread im [Forum](https://forum.dcvault.net). So klären wir die Richtung, bevor du Arbeit investierst.

## Sprachen

DCVault gibt es derzeit auf Englisch (der Hauptsprache) und Deutsch, und neue Sprachen sind willkommen. Englische Seiten liegen in `docs/`, jede Übersetzung unter `i18n/<locale>/`.

Für eine neue Sprache öffne zuerst ein Issue oder einen [Forum](https://forum.dcvault.net)-Thread. Ein Maintainer trägt die Sprache in die Konfiguration ein, danach kannst du Seite für Seite übersetzen: Die Dateien spiegeln `docs/` unter `i18n/<locale>/docusaurus-plugin-content-docs/current/`. Du musst nicht alles auf einmal übersetzen.

Wenn du auf Englisch oder Deutsch arbeitest, trag die Sprache bei, die dir liegt, und weise darauf hin, was noch fehlt. Ein Maintainer hilft.

## Stil in Kürze

- Einfach und konkret. Lieber kurz als lang.
- Überschriften in Sentence-case.
- Auf verwandte Seiten verlinken statt zu wiederholen.
