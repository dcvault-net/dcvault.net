Operators keep a hub running smoothly. They enforce the rules, help newcomers, and deal with spam and abuse.

## Operators and user classes

Hubs have user classes with different rights: regular users, registered users, operators, and the owner. Operators can do more than regular users, and the exact powers depend on the software.

## Common tools

- Kick or ban a user, by nick or by address.
- Manage rights: register users, promote operators, set per-class permissions.
- Set and show the rules in the hub topic or a main-chat message.
- Use scripts or built-in features to catch spam and flooding.

## A healthy hub

Clear, short rules and consistent moderation matter more than heavy-handed tools. Be present, be fair, and explain decisions when you can. See [Scripting](/docs/hub/scripting) for automating the repetitive parts.
