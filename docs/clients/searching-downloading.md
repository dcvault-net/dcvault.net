Once you are on a hub you can browse what others share and download straight from them.

## Browsing a file list

![Opening a user's file list in DC++](/img/getting-started/7_filelistandbrowse.png)

1. Select a user and right-click to open the context menu.
2. Choose "Get file list" to see what they share.
3. A new window opens with their files, which you can browse.

## Downloading

![Downloading from a file list in DC++](/img/getting-started/8_downloading.png)

1. Browse the share to find the file or folder you want.
2. Right-click the folder and choose "Download" to start.
3. The transfer window shows the progress of your downloads.

Downloads come straight from the other user, not the hub. If transfers stall, check [active vs. passive mode](/docs/clients/active-vs-passive-mode).
