There are a few clients still actively developed. Which one you use is largely personal preference. The table lists clients currently under active development plus a few popular ones.

| Client | Windows | Linux | macOS | Interface | License | Status | Download |
| --- | --- | --- | --- | --- | --- | --- | --- |
| DC++ | Yes | No | No | GUI | GPL-3 | Active | [Download](https://dcplusplus.sourceforge.io/) |
| AirDC++ | Yes | Yes | No | GUI / Web UI | GPL-3 | Active | [Download](https://airdcpp.net/docs/installation/installation.html) |
| FearDC | Yes | No | No | GUI | GPL-3 | Active | [Download](https://client.feardc.net/) |
| Ncdc | Yes | Yes | Yes | CLI / TUI | MIT | Active | [Download](https://dev.yorhel.nl/ncdc) |
| EiskaltDC++ | Yes | Yes | Yes | GUI | GPL-3 | Inactive | [Download](https://github.com/eiskaltdcpp/eiskaltdcpp#packages-and-installers) |
| ApexDC++ | Yes | No | No | GUI | GPL-2 | Inactive | [Download](https://www.apexdc.net/download/) |

Other clients exist but have not been updated in years. There is also a [comparison on Wikipedia](https://en.wikipedia.org/wiki/DC++#Client_software_comparison).

Once you have picked one, head to [Installation & setup](/docs/clients/installation-setup).
