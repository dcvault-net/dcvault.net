New to it? Direct Connect (DC++) is a peer-to-peer network for sharing files and
chatting on community-run hubs. Every topic lives in the sidebar, and this page
points you to a good place to begin.

## Where do I start?

[What is Direct Connect?](/docs/basics/what-is-direct-connect) is the five-minute
intro if you are brand new. If you would rather dive in, the
[Quick start](/docs/basics/quick-start) gets you onto the network in a few steps.

## Use the network

Set up a client, find hubs, and share files.
[Get started as a user](/docs/clients/choosing-a-client).

## Run a hub

From choosing hub software to scripting.
[Get started as a hub operator](/docs/hub/basics).

## Join the community

Questions and discussion live on the [forum](https://forum.dcvault.net). Spot
something missing or wrong? [Anyone can contribute](/docs/community/contributing),
and this wiki grows with the community.
